# Data Mining Practice

This repository contains exercises for the **Data Mining course** implemented using **Python in VS Code**.

## Project Structure

data-mining-practice/
│
├── practice1_eda
│   Exploratory Data Analysis with Titanic and Iris datasets
│
├── practice2_machine_learning
│   Regression and clustering models
│
└── practice3_classification
Customer churn prediction using machine learning models

## Practice 1 – Exploratory Data Analysis (EDA)

Dataset:

* Titanic
* Iris

Tasks:

* Load and explore dataset
* Check missing values
* Data normalization
* Encoding categorical variables
* Visualization (Histogram, Boxplot)
